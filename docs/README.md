# AWS Docs

AWS 학습 문서는 서비스와 운영 주제별로 나눠 관리합니다.

| 폴더 | 내용 |
|------|------|
| `ec2/` | EC2 인스턴스, AMI, EBS, LVM 마운트, Spot, SSM, UserData |
| `eks/` | EKS 네트워킹, 노드그룹, IRSA, Karpenter, 업그레이드 |
| `network/` | VPC, TGW, Route 53, VPC Endpoint, Flow Logs |
| `security/` | IAM, IMDS, CloudTrail, WAF, 보안 감사 |
| `cloudwatch/` | CloudWatch Logs, Metrics, Alarm, Dashboard, RUM |
| `storage/` | S3, EFS, ECR 운영 패턴과 스토리지 비용 최적화 |
| `database/` | RDS, Aurora, ElastiCache 운영 |
| `dx/` | Direct Connect, BGP, VIF, 회선 모니터링 |
| `cost/` | 비용 최적화, Organizations, 멀티 계정 |
| `platform/` | AWS CLI 내부, CodeDeploy, Landing Zone |
| `troubleshooting/` | 장애 분석과 진단 문서 |
| `agents/` | AI 전문 에이전트 프롬프트 |
| `rules/` | 문서 작성과 AWS 운영 규칙 |
| `templates/` | 재사용 문서 템플릿 |

실행 가능한 예제와 자동화 코드는 `../ops/README.md`를 참고합니다.
